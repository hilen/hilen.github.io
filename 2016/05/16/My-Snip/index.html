Mac 下多个 SSH Key 配置方法

【1】配置 git user.name， user.email

【2】生成 SSH Key
打开终端，并输入 ssh-keygen -t rsa -C "hilenkz@gmail.com" -f hilen_rsa

上面的 hilen_rsa 就是文件名，这时 ~/.ssh 目录下会多出 hilen_rsa 和 hilen_rsa.pub 两个文件， hilen_rsa.pub 里保存的就是我们要使用的key。

【3】配置 SSH config 文件
打开　~/.ssh/config （如果没有就创建一个）

在 config 文件内输入你自己的配置

【4】查看配置是否生效
终端内输入： ssh-add -l

查看是否在 SSH 列表内，如果不在，执行：ssh-add -K ~/.ssh/hilen_rsa

【5】测试 SSH Key 是否配置成功
终端：ssh -T git@github.com

证明成功了

【6】复制你的 publickey
pbcopy < ~/.ssh/hilen_rsa.pub 复制这个 key, 粘贴到你要的位置。OK done

iTerm 的快捷键使用

快捷键 作用
⌘ 数字 在各 tab 标签直接来回切换i
⌘ f 查找
⌘ ; 自动完成
⌘ d 横着分屏
⌘ ⇧ d 竖着分屏
⌘ ⇧ H 命令历史
⌘ ⌥ B 命令回放
⌃ a 光标移动到行首
⌃ e 光标移动到行末
⌃ f 前移一个字符
⌃ b 后退一个字符
⌃ l 清屏
⌃ p 显示历史命令
⌃ r 倒转查找
⌃ d 删除当前光标的字符
⌃ h 删除光标之前的字符
⌃ w 删除光标之前的单词
⌃ k 删除光标之后整行
⌃ u 清空当前行，无论光标在什么位置

键位说明：

  • Command ⌘
  • Shift ⇧
  • Option ⌥
  • Control ⌃
  • Caps Lock ⇪

安装

从 https://www.sublimetext.com/3 下载并且拖拽到 Application 文件夹内。

Sublime Text 比较强大的就是它的插件了，那么这里就要提到 Package Control 这个插件管理包了。
安装方法： 使用 Ctrl + ` 快捷键或者通过 View -> Show Console 菜单打开命令行。然后输入

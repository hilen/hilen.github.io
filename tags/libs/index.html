Instagram iOS App Libraries

11月 04, 2015 发布在 Objective-C

以下表格是 Instagram iOS App 所用到的第三方库，具体协议内容可以在这里找到。

名称 描述
AFNetworking 这个不用介绍了把
Appirater 提醒用户评价 App 的组件
Apple Reachability 判断网络状态
AVCamCaptureManager 用来录制视频
Boost Boost 库是为 C++ 语言标准库提供扩展的一些 C++ 程序库的总称。
CCNode-Screenshot 截屏组件
CocoaHTTPServer 可以在 Mac OS X 或iOS应用内嵌入 Server
CocoaLumberjack 一个快速、简单,但很强大的日志框架,用于 Mac OS X 和 iOS 系统
cocoawithlove-Base64 使用 base64 对 data 进行编码
ColorTracking 看标题吧
Flickr-OAuth-iOS 这个开源的iOS库允许你的 Flickr 集成到你的 iOS 应用程序
GLTextureAtlas 官方 demo 展示用 texture atlas 和 OpenGL ES 渲染多个具有不同的纹理的对象
iOS-OpenGLES-Stuff OpenGL ES 编程的 iOS 实例
ios5-cookbook 这本iOS5开发教程的源码
JSONKit Objective-C 的JSON的解析库
LXReorderableCollectionViewFlowLayout UICollectionViewFlowLayout 的扩展，支持重新排序
KTOneFingerRotationGestureRecognizer 实现了自定义的 UIGestureRecognizer 用于实现单手指旋转操作，同时会跟踪手指的移动
MBProgressHUD 很多种样式的提示框
MyOpenALSupport OpenAL
NSNotifications and Background Threads NSNotifications and Background Threads
NSString+XMLEntitites 解析 XML，文件在链接的 Class 内
oauthconsumer iOS OAuth consumer
picojson C++ 的JSON解析库
PLCrashReporter 可以记录应用crash的框架
QSUtilities 提供一些常用功能的库（字符串清理，网络接入等）
scifihifi-iphone Keychain安全保存用户密码
SocketRocket 一个 WebSocket 的第三方库
soundswitch 切换播放设备库（声卡）
XBImageFilters 使用OpenGL ES 2 来快速处理各种图片效果
zxtools 各种工具类
